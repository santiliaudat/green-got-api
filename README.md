# GREEN-GOT API

### Endpoints

| Method | URL                          | Params              | Notas             |
| ------ | ---------------------------- | ------------------- | ----------------- |
| GET    | /api/greetings/${first_name} | first_name          | Return a greeting |
| POST   | /api/create_user             | firstName, lastName | Create user       |

### Getting Start Development

```bash
git clone https://github.com/xoolix/admin.git
cd green-got-api
yarn
yarn dev
```

### Run tests

```bash
git clone https://github.com/xoolix/admin.git
cd green-got-api
yarn
yarn test
```
